# Alura-projeto-alugando-games-
Logica simples de uma locadora

**Esse é um projeto que fiz na Alura para treinar lógica de programação** e manipulação do DOM com JavaScript. O objetivo do sistema é controlar o aluguel e a devolução de jogos em uma página.

## O que o projeto faz:
* Altera visualmente o botão e a imagem do jogo quando ele é alugado ou devolvido.
* Gerencia o status de cada jogo dinamicamente usando funções no JavaScript.

## **Minhas Modificações:**
* **Adicionei integração com o usuário:** agora o sistema pede o nome da pessoa através de um prompt assim que ela entra na página e exibe uma mensagem personalizada de boas-vindas.
* **Criei uma trava de segurança no aluguel:** coloquei um limite máximo onde a pessoa não consegue alugar mais de dois jogos ao mesmo tempo. Se ela tentar pegar os 3 jogos, o sistema avisa que ela precisa devolver pelo menos 1 para poder continuar.

## Tecnologias usadas:
* HTML5
* CSS3
* JavaScript
